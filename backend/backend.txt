Authentication

For a social media application:

Access Token + Refresh Token

Use:

JWT Access Token (15 mins)
Refresh Token (30 days)

Store:

refresh_tokens

table in database.

Access token:

Authorization: Bearer token

Refresh token:

HttpOnly Cookie

This is currently one of the most common architectures.

Password Hashing

Use:

npm install argon2

Instead of bcrypt.

Argon2 is considered stronger and more modern.
